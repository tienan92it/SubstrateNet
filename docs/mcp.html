MCP integration

25 tools over stdio — code (L0/L0.5), knowledge (L1.5–L3), domain (L2.5), the global skill view (L4–L5), and a research surface (ask, requirements, incidents, workspace). Query your cross-project graph from any MCP-aware agent.

Cursor

Add to ~/.cursor/mcp.json:

{
  "mcpServers": {
    "subnet": {
      "command": "subnet",
      "args": ["serve", ".", "--mcp"]
    }
  }
}

Replace "." with an absolute project path if you want the server to operate on a fixed project regardless of where Cursor is invoked.

Claude Code

Add to ~/.claude/claude_desktop_config.json (macOS path may vary):

{
  "mcpServers": {
    "subnet": {
      "command": "subnet",
      "args": ["serve", "--mcp"]
    }
  }
}

Codex CLI / other

Any MCP-compatible client that supports the stdio transport works the same way. The command is subnet serve [path] --mcp; no flags beyond --mcp.

Usage patterns

Carrying decisions forward

When starting work on a new feature, prompt the agent to recall prior decisions before writing code:

"Before designing the payment retry flow, call subnet_decisions with topic 'retry' and tell me what we've already decided. Then call subnet_business_logic for 'payment' to surface invariants."

Onboarding a teammate

subnet_concepts with no arguments lists the top concepts in a project ranked by member count. subnet_explain conceptId expands one into a structured brief.

Auditing what the system kept

If a decision you remember discussing didn't surface, run subnet_triage_audit droppedOnly:true to see what got filtered and why. The rationale string captures the Triage Agent's reasoning; if the filter was wrong, retune triage (bump prompt version or swap model).

Transport details

Stdio only. The server logs to stderr (not visible to the MCP client) and speaks JSON-RPC 2.0 on stdout. No network sockets are opened.

Verify the server independently: 

printf '{"jsonrpc":"2.0","id":1,"method":"initialize","params":{"protocolVersion":"2024-11-05","capabilities":{},"clientInfo":{"name":"test","version":"0"}}}\n{"jsonrpc":"2.0","id":2,"method":"tools/list"}\n' \
  | subnet serve . --mcp
