Cross-project knowledge, distilled to wisdom.

Substrate Net reads your code and AI-agent conversations and builds a grounded, queryable picture of what you know across every project — organized as a DIKW pyramid that ends in Wisdom: the competencies you've mastered, the insights your work reveals, and the gaps worth closing next. All local, all grounded.

$ subnet → architecture dashboard source
WISDOM leveled competencies · cross-project insights · gaps to close KNOWLEDGE concepts · business + tech domains · the skill graph INFORMATION decisions · rules · requirements · incidents — triaged DATA your code · AI-agent transcripts · in-repo docs & diagrams

Climbs to wisdom.

Each tier is built from the one below. The top is a synthesized professional profile: competency areas graded novice → expert, insights, and the gaps to close.

Grounded, never assumed.

Every claim carries its source — structural, stated, corroborated, external, or model — so project truth and inference never blur. Synthesis stays clearly tagged as inference.

Local + hybrid.

Tree-sitter resolves exact structure; LLMs read it to produce meaning — fewer tokens, fewer hallucinations. Everything runs on local SQLite, no keys required.

Minimal Run

# 1. Install (Node 20+)
git clone https://github.com/tienan92it/SubstrateNet && cd SubstrateNet
npm install && npm run build:all && npm link

# 2. Backend (optional). Flash-first when OPENROUTER_API_KEY is set; otherwise
#    fully local via Ollama — every agent falls back automatically.
ollama pull qwen2.5:14b
ollama pull nomic-embed-text

# 3. First run: discover → estimate → full pipeline → dashboard
subnet setup --plan-only   # optional per-phase cost table
subnet setup

# 4. Day to day (or just run `subnet` for the menu)
subnet update              # standard: tier-1 analyze + fused enrich
subnet update --fast       # lean: transcripts only

# 5. See what you know
subnet global wisdom            # leveled competencies, insights, gaps
subnet global dashboard --open  # the DIKW profile + knowledge map
